# opal-proto

An **experimental, unbranded prototype** — a test harness for an opal-cutting interaction loop. Not a product, not affiliated with any brand. Throwaway code for iterating on game-feel and feedback design.

**Play it:** https://empireoftrees.github.io/opal-proto/

You get a procedurally generated rough with a hidden opal band at varying depth. Read the cues, grind to the band with no undo, and try to get as close to **Par** (ideal play) as possible. The score breakdown is fully visible. Tap **Copy result** to share exact state for feedback.

This is auto-deployed from a private R&D workspace; expect frequent, breaking changes.
